# message_generator

## Description
This is my version of the Mixed Messages project on Codecademy.

## Technologies
Project uses:
* Node: v14.15.4

## Setup
Make sure you have node installed on your machine: 
https://nodejs.org/en/

To run my project:
```bash
$ git clone https://github.com/L1v1on/message_generator.git
$ cd message_generator/
$ node script.js
```
